# Merkle Airdrop



This is an implementation of a private Merkle Tree based airdrop built using Mina smart contracts. 

# Installation
**Note: these commands do not work with `yarn`**
## How to build



```sh
npm run build
```

## How to run tests

```sh
npm run test
```

## How to run coverage

```sh
npm run coverage
```
